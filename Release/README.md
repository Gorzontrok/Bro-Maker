# BroMaker

BroMaker allows you to load custom bro mods into the game and provides modders with a library to greatly decrease the difficulty of adding custom bros to the game.

## Installing BroMaker and Custom Bros

Detailed installation instructions can be found [here](https://steamcommunity.com/sharedfiles/filedetails/?id=2434812447).

## Bugs and Issues

If you run into any bugs or have any issues with the mod, please join the [Free Lives Discord](https://discord.com/invite/freelives) and post in the bf-mods channel.

To play with a Custom Bro, click on it in the BroMaker mod GUI, and click Load Bro. Or you can enable automatic spawning and set a chance to spawn with a custom bro in the mod settings, by default custom bros have the same chance to spawn as normal bros.
